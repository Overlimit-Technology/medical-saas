# README - 29-01-2026

Este documento resume lo que se hizo hoy, cómo se implementó y cómo configurar las conexiones del proyecto.

## Qué se hizo
- Calendario de citas interactivo: selección por arrastre para crear citas, modal elegante, validación de solapamientos, indicador de “hoy”, navegación semanal y citas en bloques visuales.
- CRUD de pacientes 100% en base de datos: crear, leer, actualizar y eliminar usando la API `/api/patients`, con validaciones (RUN chileno, email, teléfono), búsqueda y feedback visual.

## Cómo se hizo (resumen técnico)
### Calendario de citas
- Interacción de arrastre con `pointer` para seleccionar rangos de tiempo en la grilla semanal.
- Vista semanal 8:00–20:00 con slots de 15 minutos.
- Modal con formulario y validación de superposición antes de guardar.
- Estado visual: bloque turquesa mientras arrastras, citas existentes como tarjetas, línea de “ahora” en el día actual.

Archivos clave:
- `src/app/(app)/agenda/page.tsx`
- `src/app/globals.css` (animación `fade-in`)

### Pacientes (CRUD completo en DB)
- El frontend consulta `/api/patients` para listar, crear, editar y eliminar.
- Se eliminaron los datos en `localStorage` (todo va a la BD).
- Validaciones front-end y mensajes de error amistosos.

Archivo clave:
- `src/app/(app)/patients/page.tsx`

## Cómo configurar conexiones (base de datos)
Este proyecto usa Postgres + Prisma.

1) Crear el archivo `.env.local` con la URL de conexión:
```
DATABASE_URL="postgresql://USER:PASSWORD@HOST:PORT/DBNAME"
```

2) Generar Prisma Client:
```
npx prisma generate
```

3) Aplicar migraciones (si es necesario):
```
npx prisma migrate dev
```

4) Levantar el proyecto:
```
npm run dev
```

## Notas de funcionamiento
- Si aparece un error del tipo “Cannot read properties of undefined (reading 'findFirst')”, suele indicar que Prisma Client está desactualizado o no generado. Ejecuta `npx prisma generate` y reinicia el servidor.
- Los pacientes están asociados a una clínica. Asegúrate de tener sesión y clínica seleccionada para poder ver/crear datos.
