## Estándares HL7 (resumen)
- HL7 v2: mensajería basada en segmentos (p. ej., ADT/ORM/ORU).
- HL7 FHIR: API basada en recursos (REST/JSON).
- HL7 CDA: documentos clínicos estructurados.
